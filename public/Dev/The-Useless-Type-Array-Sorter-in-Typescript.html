    The first important thing is to create a loop with stop condition. It seems very simple, but there‚Äôs a problem in Typescript : programming with types is kind of functional programming and not imperative programming. Writing a while would be something like this :

    But I said kind of functional programming. Because Typescript can compute basic types (boolean, string, etc.) and generic types (ReturnType, Omit, etc.) but can‚Äôt compute generic of generic types. If we compare with functional programming, we get this :

    TypescriptFunctional programming language
